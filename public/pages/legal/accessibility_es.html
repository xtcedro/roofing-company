Declaración de Accesibilidad

Nuestro compromiso para garantizar que nuestro sitio web sea accesible para todos.

Fecha de Vigencia: 1 de enero de 2024

1. Nuestro Compromiso con la Accesibilidad

En Heavenly Roofing, estamos comprometidos a proporcionar un sitio web accesible para todos, independientemente de sus habilidades o discapacidades. Nuestro objetivo es garantizar que todos los usuarios tengan una experiencia fluida al navegar por nuestro sitio web y acceder a nuestros servicios.

2. Estándares de Accesibilidad

Este sitio web ha sido diseñado para cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1, Nivel AA. Estas pautas ayudan a garantizar que el contenido sea accesible para un rango más amplio de personas con discapacidades, incluidas aquellas con ceguera, baja visión, pérdida auditiva y discapacidades cognitivas.

3. Características de Nuestro Sitio Web Accesible

  • Alternativas de texto para contenido no textual.
  • Navegación por teclado para todos los elementos interactivos.
  • Relaciones de contraste que hacen que el texto y los elementos visuales sean legibles.
  • Diseño responsivo que se adapta a diferentes tamaños de pantalla y tecnologías asistivas.

4. Mejoras Continuas

Estamos trabajando continuamente para mejorar la accesibilidad de nuestro sitio web. Los comentarios de nuestros usuarios son vitales para ayudarnos a lograr nuestros objetivos.
